# FileRouter blog authoring

Add a `.md` file to this directory for each post. The filename becomes the URL
slug.

```md
---
title: Choosing a document parser
description: A practical framework for comparing document parsing providers.
date: 2026-07-20
author: FileRouter
category: Guides
draft: true
---

Start writing here.
```

Valid categories are `Product`, `Guides`, `Engineering`, and `Company`. Set
`draft: false` when the post is ready to appear on the blog, in RSS, and in the
sitemap.
